# Microsoft

## 概述
全球科技巨头，拥有GitHub、开发AutoGen框架，投资AI Agent开发，在金融分析和AI Agent架构讨论中被用作案例研究对象。

## 基本信息
- 类型：公司
- 相关领域：云计算、AI、软件开发

## 在AI生态中的角色
- 开发AutoGen Agent框架
- 提供"AI Agents for Beginners"课程
- 拥有GitHub代码托管平台
- 云业务增长和AI战略案例研究对象

## 近期动态
- 投资GitHub前CEO创建的替代项目（6000万美元）
- 投资自己的竞争对手产品
- 模型接入支持GPT、DeepSeek等
