# awesome-claude-code

## Description
A popular GitHub repository that serves as a curated list of tools, plugins, skills, workflows, and resources built around the [[Claude Code]] AI coding assistant. With over 31,000 stars, it is a central hub and definitive signal of the growing **[[AI Programming Ecosystem]]** surrounding Claude Code.

## Purpose and Significance
- **Ecosystem Manifestation**: The repository's existence and popularity demonstrate that developers are treating Claude Code as a *platform* for building upon, not just a tool to use.
- **Community Knowledge Base**: It aggregates community-discovered best practices, prompt patterns, and integrations, accelerating onboarding and mastery.
- **Switching Cost**: By providing immense added value, it increases user investment in the Claude Code ecosystem.

## Contents
The repository typically includes categories such as:
- **Tools & Applications**: Third-party desktop or web applications that enhance the Claude Code experience, such as **[[Clave]]**.
- **Skills & Slash Commands**: Pre-configured prompts and command sets for specific tasks or workflows, including formalized systems like **[[GStack: Role-Based AI Development Workflow for Claude Code]]**.
- **Integrations**: Guides and code for connecting Claude Code with other IDEs, terminals, or services.
- **Workflows & Tutorials**: Shared methodologies for effective AI-assisted development.
