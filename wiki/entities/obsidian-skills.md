# obsidian-skills

## Overview
An open-source toolkit by [[Obsidian]]'s CEO [[Steph Ango (kepano)]] containing five skills that allow AI agents like Claude Code to generate native Obsidian content and structures. Represents a bridge between AI capabilities and Obsidian's content model.

## Core Skills

### obsidian-markdown
- **Native Format Generation**: Creates properly formatted Obsidian Markdown.
- **Extended Syntax**: Supports Obsidian-specific features like wikilinks and callouts.
- **Structure Preservation**: Maintains consistent heading hierarchies and formatting.

### json-canvas
- **Canvas Creation**: Generates Obsidian Canvas files (JSON-based visual workspace).
- **Node Placement**: Positions content nodes with coordinates and connections.
- **Visual Layouts**: Creates aesthetically arranged knowledge maps and diagrams.

### obsidian-bases
- **Structured Data**: Generates content with consistent property schemas.
- **Database-like Features**: Creates queryable structured notes.
- **Template Integration**: Works with Obsidian's templating system.

### Additional Skills
- **Content Transformation**: Converting between formats within Obsidian's ecosystem.
- **Metadata Management**: Handling frontmatter and note properties.
- **Link Management**: Creating and maintaining connections between notes.

## Technical Implementation
- **Open Source**: Available on GitHub for community contribution and inspection.
- **AI Agent Integration**: Designed for use with Claude Code, OpenClaw, and similar agents.
- **Tool Chaining**: Skills can be combined for complex workflows.

## Applications
- **[[AI-Enhanced Note-Taking]]**: Automates note creation and formatting.
- **Knowledge Base Population**: Rapidly generates structured content for new topics.
- **Template Instantiation**: Fills predefined note templates with AI-generated content.
- **Visual Knowledge Mapping**: Creates canvas diagrams from textual descriptions.

## Philosophical Significance
- **[[Markdown as Human-AI Interface]]**: Implements this concept practically.
- **Augmentation vs. Replacement**: Designed to enhance rather than replace human note-taking.
- **Structured Collaboration**: Provides clear boundaries for AI contribution to knowledge bases.

## Related Concepts
- **[[AI Tool Chaining/Combination]]**: The toolkit exemplifies tool chaining by allowing different AI agents to leverage the same skills.
