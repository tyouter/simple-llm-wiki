**Obsidian Dragger** is a community-developed plugin for [[Obsidian]] that enables Notion-like drag-and-drop rearrangement of text blocks on both desktop and mobile interfaces. Created by [[Ariestar]], the plugin addresses a significant gap in Obsidian's native editing experience, particularly the difficulty of reorganizing content on touchscreen devices.

The plugin implements the [[Text Block as a Unit of Manipulation]] paradigm, treating paragraphs, lists, tasks, tables, and callouts as discrete, movable objects. It features comprehensive [[Multi-Format Block Support]], ensuring diverse content types maintain their formatting and functionality when dragged. This directly enhances [[Content Structure Agility]], allowing users to rapidly experiment with information architecture.

Currently in beta development, Obsidian Dragger is distributed via [[BRAT (Beta Reviewer's Auto-update Tool)]], exemplifying how the [[Obsidian Plugin Ecosystem]] facilitates early testing and community feedback. The plugin represents [[Paradigm Borrowing in UX]] by importing [[Notion]]'s successful block manipulation model into Obsidian's more file-centric environment. Its development story perfectly illustrates [[Community-Driven Feature Development]], where a user's personal pain point becomes a community solution.
