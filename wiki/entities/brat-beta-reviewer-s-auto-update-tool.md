**BRAT (Beta Reviewer's Auto-update Tool)** is a specialized [[Obsidian]] plugin that facilitates the installation and updating of beta or in-development community plugins directly from their GitHub repositories. It serves as an alternative distribution channel to the official community plugin marketplace, which typically requires stable, production-ready releases.

BRAT is critical infrastructure within the [[Obsidian Plugin Ecosystem]], enabling [[Community-Driven Feature Development]] by allowing developers to share work-in-progress with early testers. For plugins like [[Obsidian Dragger]] that are in active beta development, BRAT provides a seamless installation experience with automatic updates as the developer pushes new commits.

This tool lowers barriers for both developers (who can gather feedback earlier) and users (who can access cutting-edge features sooner). It represents a pragmatic approach to quality assurance in open ecosystems, recognizing that community testing often identifies edge cases and usability issues that escape controlled development environments. BRAT exemplifies how thoughtful tooling can accelerate innovation cycles in community-driven software ecosystems.
