# Dataview

## Overview
An [[Obsidian]] plugin that enables querying and displaying data from notes using a JavaScript-like syntax, used for creating dynamic indexes, summaries, and aggregated views of knowledge base content.

## Core Functionality

### Query Language
- **JavaScript-like Syntax**: Familiar syntax for developers and technical users.
- **Note Properties**: Query frontmatter (YAML metadata) and inline fields.
- **Content Analysis**: Search and filter based on note content, tags, and links.

### Display Options
- **Tables**: Create sorted, filtered tables of notes meeting specific criteria.
- **Lists**: Generate dynamic lists with custom formatting.
- **Task Views**: Aggregate tasks from across the vault with filtering.
- **Calendar Views**: Display notes chronologically.

## Use Cases

### Knowledge Management
- **Dynamic Indexes**: Automatically updated lists of notes by category, tag, or property.
- **Progress Tracking**: Monitor completion status of projects or learning goals.
- **Content Discovery**: Find related notes based on metadata and content.

### Project Management
- **Task Aggregation**: Collect tasks from project notes into single views.
- **Status Dashboards**: Create overviews of project progress and next actions.
- **Meeting Notes**: Link and summarize meeting notes across projects.

### Research Organization
- **Literature Reviews**: Aggregate and categorize research papers and notes.
- **Source Management**: Track references, citations, and reading progress.
- **Concept Maps**: Show relationships between research topics.

## Integration with Organizational Approaches
- **[[Note Organization Philosophy]]**: Dataview enables dynamic organization that complements or replaces static structures.
- **Metadata-Driven Design**: Encourages consistent use of frontmatter and properties.
- **Hybrid Systems**: Combines folder hierarchy with query-based views.

## Advanced Features
- **Inline Queries**: Embed queries directly within notes.
- **Custom Functions**: Extend functionality with user-defined JavaScript.
- **Live Updates**: Views update automatically as notes change.
- **Export Options**: Output query results in various formats.

## Community and Ecosystem
- **Popular Plugin**: One of the most widely used Obsidian plugins.
- **Learning Resources**: Extensive documentation and community tutorials.
- **Related Plugins**: Works with other plugins for enhanced functionality.

## Related Concepts
- **[[Note Organization Philosophy]]**: Dataview represents a query-based approach to organization.
- **[[AI-Enhanced Note-Taking]]**: AI can generate Dataview-compatible metadata and queries.
- **[[Git-Based Knowledge Management]]**: Dataview queries work well with version-controlled vaults.
