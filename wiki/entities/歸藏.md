# 歸藏

## Overview
Chinese developer who created the [[CodePilot]] project, an open-source implementation of Claude's [[Generative UI]] functionality for AI-generated charts.

## Technical Contributions
### [[CodePilot]] Implementation
Developed an alternative architecture for [[Generative UI]] using a [[Code Fence + Sandbox Iframe Architecture]] instead of Claude's official `tool_use` mechanism. Key innovations include:
- Real-time [[Streaming Rendering]] of charts during generation
- Support for multiple AI models including [[Kimi K2.5]] and [[MiniMax M2.5]]
- Sandboxed execution for security

### Performance Claims
Assertions made in the [[Zhihu Discussion: Open Source Projects for AI Visualization and Hardware Development]]:
- Superior streaming experience compared to Claude's implementation
- [[Kimi K2.5]] can produce better visualizations than some established models
- The alternative architecture allows preview during code generation

## Development Philosophy
Emphasizes practical implementation and user experience in AI visualization tools. Represents the trend of Chinese developers creating specialized tools within the [[AI Tool Specialization for Design]] domain.

## Context within Chinese Developer Community
Part of the ecosystem discussed in [[2026 Chinese AI Coding Plan Comparison and NVIDIA NIM Integration]], utilizing models available through various coding plans and API services.
