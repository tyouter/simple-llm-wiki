# Salvatore Sanfilippo (antirez)

Redis 数据库的创造者，以追求极致性能和简洁代码的工程哲学闻名于开源社区。

## 核心要点

- **Redis 创造者**: 2009年创造了 Redis，这个内存键值存储系统成为现代 Web 应用的基础设施组件
- **性能优先**: 坚持 C 语言实现，追求极致性能和资源效率，Redis 成为缓存系统的行业标准
- **简洁代码哲学**: 以"代码应该像散文一样可读"为理念，Redis 源码被誉为教科书级别的清晰代码
- **个人博客**: 在 antirez.com 分享技术见解和创业思考，文章风格深入、坦诚
- **开源精神**: 期独立维护 Redis，体现了个人开发者对开源项目的深度投入

**Salvatore Sanfilippo**， 网络昵称antirez，是内存数据库Redis的创始人。他是一位充满激情、表达欲旺盛的意大利程序员，以其在技术上的极简主义设计哲学和直率的个人风格而闻名。与许多顶级开源项目维护者一样，他拥有并持续更新着自己的[[独立个人博客价值|个人博客]]。

他的博客内容混合了技术思考（尤其是关于数据库、系统设计和编程语言）、项目日志、对行业事件的评论以及个人随笔。近期，他的博客也出现了[[AI技术博客主题聚合|明显的AI主题聚合]]，但他采取了一种独特的方式：经常发布由AI工具（如LLM）对他感兴趣的较长文章或讨论进行的总结，并附上他自己的犀利评论和看法。这种方式本身就是在探索和展示人类与AI在信息处理上的协作模式。antirez的博客是观察一位天才程序员如何以个人视角解读世界，并积极参与[[AI-Induced Industry Disruption|当前技术范式变革]]的生动窗口。
