# GitHub

## 概述
全球最大的代码托管平台，被微软收购，是开源项目的核心托管地，也是AI工具和项目的主要发布平台。

## 基本信息
- 类型：平台/公司
- 所属：Microsoft（2018年收购）
- 相关领域：代码托管、开源协作、版本控制

## 在AI生态中的角色
- AI开源项目的主要托管平台
- AI工具系统提示词泄露项目获得122K Star
- Claude Code Superpowers等插件发布地
- DeerFlow等Agent引擎开源地

## 近期动态
- GitHub前CEO创建替代项目，获微软投资
- "Git时代结束了"的说法引发讨论
- AI时代开发工具重新定义的尝试
