# GitHub

## 概述
全球最大的代码托管平台，被微软收购，是开源项目的核心托管地，也是AI工具和项目的主要发布平台。

## 基本信息
- 类型：平台/公司
- 所属：Microsoft（2018年收购）
- 相关领域：代码托管、开源协作、版本控制

## 在AI生态中的角色
- AI开源项目的主要托管平台
- AI工具系统提示词泄露项目获得122K Star
- Claude Code Superpowers等插件发布地
- DeerFlow等Agent引擎开源地

## 近期动态
- GitHub前CEO创建替代项目，获微软投资
- "Git时代结束了"的说法引发讨论
- AI时代开发工具重新定义的尝试

**GitHub** 是全球最大的开源代码托管和协作平台。在本文语境中，它是 **[[TypeWords]]** 项目源代码的托管地，是项目“开源”属性的技术基础。平台为开发者提供了版本控制、问题追踪、协作请求等功能，是 **[[AI Programming Ecosystem]]** 的核心组成部分。

对于TypeWords这类教育工具项目，GitHub不仅是一个代码仓库，也是项目展示、吸引开发者贡献以及践行 **[[开源学习工具的可试先行]]** 策略的起点。项目维护者可以在GitHub仓库中提供项目描述、在线Demo链接和部署指南，引导用户从“查看代码”到“体验产品”。平台的存在极大地降低了优秀学习工具项目的传播和获取门槛。

## 概述
全球领先的软件开发和版本控制托管平台，基于Git。它是开源项目和协作开发的核心枢纽。

## 在本框架中的角色
在[[AI决策五步法：提升判断与决策能力的结构化框架]]的语境下，**GitHub**是作者将其实践方法**工程化**和**开源**的发布平台。作者将“AI决策五步法”这一套方法论封装成一个具体的、可复用的“Skill”（技能包），并将其代码、使用说明或模板开源在GitHub上。这体现了现代知识工作的一种新范式：将最佳实践、工作流甚至思维框架，像软件一样进行封装、版本管理和社区共享。此举使得该方法论不再是一篇孤立的文章，而是一个可以被他人直接使用、改进和分发的“产品”，紧密关联了[[Agent Skills Framework (Applied)|智能体技能框架（应用）]]的理念。
