# designprompts.dev

## Description
A niche, specialized website that functions as a curated repository for design prompts. Its primary purpose is to aid in generating high-quality, non-generic user interface code using AI coding assistants.

## Key Features
- **Curated Collection:** Hosts a growing collection of over 20 pre-built design prompts.
- **Style Variety:** Prompts cover a range of aesthetics including Brutalist, Retro, Minimalist, Glassmorphism, and more.
- **Previewable:** Each prompt is accompanied by a visual preview, allowing users to see the general "vibe" or style it produces before use.
- **Structured Format:** Prompts are provided in a detailed XML format, designed to be fed directly into AI coding tools like [[Claude Code]].
- **Downloadable:** Users can easily download the XML files for local use and adaptation.

## Purpose in the Workflow
The site is a foundational resource in workflows aimed at [[AI-Free Interface Aesthetics]]. It provides the raw, high-quality prompts that enable techniques like [[Vibe Coding]] and [[Prompt Combination & Adaptation]]. By offering alternatives to generic AI outputs, it helps users bypass common AI visual clichés.

## Typical Use Case
A user browsing the site selects a prompt matching their desired aesthetic (e.g., "Retro Futurism"), downloads the XML, and provides it to [[Claude Code]] with instructions to generate a landing page based on that design specification. This process is detailed in [[Using DesignPrompts.dev to Generate High-Quality AI-Free Interfaces with Claude Code]].
