# 给 AI Agent 装上全网感知能力，CLI联网神器 Agent-Reach 开源，让 OpenClaw 可以自由冲浪 B 站/推特/小红书

## 概述

## 简介


AI Agent 虽然聪明，但联网获取信息一直是痛点——平台反爬、API收费、跨域封禁，每上一个平台就要踩一遍坑。Agent-Reach 正是为解决这一痛点而来：它把 X、Reddit、YouTube、B站、小红书、抖音、微信公众号等 17 个主流平台的搜索、读取能力，打包成开箱即用的 CLI 工具，已在 GitHub 收获 10。安装只需一句话，Cookie 仅存本地保障隐私，支持 Claude Code、OpenClaw、Cursor 等主流 AI Agent
