# 知乎讨论-大神们是从哪获取优质信息

## 核心内容

这是一篇关于如何获取优质信息的知乎回答汇总，提供了从入门到深入的多层次信息源分类。

### 信息源层级体系

回答者提出了清晰的信息获取层级：

1. **文献层**：Google学术搜索
2. **入门了解层**：维基百科 + 知乎 + Quora
3. **深入了解层**：垂直独立站（需入门后自然发现）
4. **新闻层**：后续网、有据核查、国际调查记者同盟

### 关键方法论

文章提出了两个核心技巧：

1. **搜索引擎语法**：利用 "+" 强制关联、"-" 排除关键词、filetype、site、intitle等高级语法
2. **阅读官方文档**：官方文档蕴含大量专业知识且通俗易懂
