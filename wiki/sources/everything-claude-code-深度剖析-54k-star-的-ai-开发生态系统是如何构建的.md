## 简介


在 GitHub 上，everything-claude-code 项目获得了超过 54,000 个 star，它不是一个简单的配置文件，而是一个完整的 AI 辅助开发生态系统。本视频将深度剖析这个项目的架构设计，从底层规则系统到 13 个专业 Agent 的协作机制，从 56+ 个领域技能到持续学习系统，带你理解它如何让 Claude Code 变得更强大。涵盖内容：
📋 双层规则体系：通用规则 + 语言特定规则
🛠️ 56+ 技能系统：结构化 Prompt 模板 + 工作流定义
🤖 13 个 Agent 协作：串行/并行/递归三种模式
🎯 32 个命令编排：一键触发复杂调用链
🔄 持续学习 v2：Instinct-based 模式识别 + AgentShield 安全审计
🔧 三层定制扩展：规则 → 技能 → Agent
适合人群：已使用 Claude Code 的开发者，想深度定制 AI 开发工作流的工程师
