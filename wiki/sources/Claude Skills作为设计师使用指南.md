# Claude Skills作为设计师使用指南

## 概述

视频由设计师Griffin Wooldridge详细介绍了Claude中针对设计师的核心Skills，包括Frontend Design、Implement Design、Theme Factory、Brand Guidelines、Canvas Design等，并演示了如何创建自定义Skill。

## 核心要点

### Frontend Design Skill
- **设计思维过程**: Claude在写代码前先进行设计思考
- **考虑因素**: Purpose、Tone、Constraints、Differentiation
- **字体指南**: 避免Inter等AI常用字体，推荐特色字体配对
- **色彩与动效**: 专业设计指导
- **反AI味**: 明确禁止"AI slop"——紫色渐变、圆角卡片、Inter字体组合

### Implement Design Skill (Figma)
- **Figma转代码**: 输入Figma URL，输出生产级代码
- **精确映射**: 读取Autolayout、Typography、Spacing等属性
- **组件复用**: 使用项目已有的组件库
- **验证清单**: Typography、Colors、Responsive、Accessibility验证
- **需要MCP**: 需连接Figma MCP服务器

### Theme Factory Skill
- **10个主题包**: 预构建的专业主题包
- **色彩字体配对**: 精心搭配的色彩方案和字体
- **多场景适用**: Slides、Documents、Web Pages、Dashboards、Reports
- **一致性**: 确保设计产出保持一致风格

### Brand Guidelines Skill
- **品牌一致性**: 自动执行品牌规则
- **模板用途**: 可作为自定义品牌Skill的模板
- **自动执行**: Colors、Typography、Spacing、Logo使用自动遵循
- **分享使用**: 可与非设计师分享使用

### Canvas Design Skill
- **视觉艺术**: 创建海报、图形设计作品
- **两步流程**: 先创建设计哲学宣言，再执行为图像
- **独特方式**: 从设计原则出发，而非从模板出发
- **PNG/PDF输出**: 输出真实图像文件

### Skill Creator
- **创建自定义**: Skill Creator Skill帮助创建新Skill
- **迭代流程**: Claude询问需求 → 写Draft → 测试 → 反馈 → 修订
- **文件夹结构**: SKILL.md文件 + 资源文件
- **分享打包**: 可打包为.skill文件分享安装
