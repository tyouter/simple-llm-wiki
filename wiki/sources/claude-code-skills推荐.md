# Claude Code Skills推荐合集

## 来源信息

- **作者**: 一直很懒的小强、老久、独元殇
- **平台**: 知乎
- **原始问题**: 你们有哪些好的claude code skills推荐？

## 推荐的Skills

### 官方/社区Skills

1. **基础开发类**
   - git操作Skill
   - 文件管理Skill
   - 代码审查Skill

2. **专项功能类**
   - PDF处理Skill
   - 数据分析Skill
   - 网页抓取Skill

3. **高级集成类**
   - MCP集成Skill
   - API调用Skill
   - 自动化测试Skill

### Skills编写建议

**一直很懒的小强**:
- 从简单功能开始
- 使用清晰的SKILL.md格式
- 包含使用示例

**老久**:
- Skills应该解决具体痛点
- 避免过度复杂化
- 注意错误处理

**独元殇**:
- 渐进披露原则
- 合理的触发条件
- 完善的文档说明

## Skills本质理解

Skills是一种"可复用的能力封装"，让Claude Code能够：
- 在特定场景自动激活
- 执行预定义的复杂任务
- 减少重复指令输入
