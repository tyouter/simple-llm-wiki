# Agent Skill vs MCP 对比分析

## 来源信息

- **作者**: 韦东东、TangLyan、傅红雪
- **平台**: 知乎
- **原始问题**: Agent Skill 为何没有像 MCP 那样火爆？

## 核心对比

### MCP协议特点

1. **标准化程度高**
   - Anthropic官方推动
   - 明确的协议规范
   - 跨平台兼容性

2. **生态优势**
   - 多个工具厂商支持
   - 社区活跃度高
   - 插件生态快速扩张

3. **应用范围**
   - 数据源接入
   - 工具调用标准化
   - 资源共享机制

### Agent Skill特点

1. **定位差异**
   - 更偏向特定场景封装
   - 内部能力复用
   - 非标准化协议

2. **传播障碍**
   - 缺乏统一规范
   - 社区碎片化
   - 跨平台移植困难

3. **实际价值**
   - 解决具体问题
   - 团队内部效率提升
   - 但难以规模化传播

## 为何MCP更火爆

关键原因：
1. **协议级标准化** - MCP是协议，Skill是实现
2. **跨生态兼容** - MCP可对接多种AI工具
3. **商业推动** - Anthropic官方背书
4. **社区效应** - 一旦形成生态便自我增长

## 未来展望

- Skills可能在特定领域深耕
- MCP作为基础设施继续扩展
- 两者可能融合而非竞争
