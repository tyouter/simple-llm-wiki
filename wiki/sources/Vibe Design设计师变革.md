# Vibe Design设计师变革

## 概述

张泽AIGC发布的视频，介绍谷歌 Studio 的重大更新——推出 Web Design 概念，像 Vibe Coding 一样说一句话就可以做设计。

## 核心变化

### 设计师天塌了

- 不是天塌了，是没了
- Google Studio 说一句话就能设计网页
- UI设计门槛真的没了

### Web Design概念

- 像 Vibe Coding 一样
- 说一句话就可以做设计
- 据说让 Figma 股价大跌8%

## 演示案例

让 Studio 设计一款博客应用：
- 面向人工智能方向
- 展示AI热点文章
- 包含各种分类：文章、图文、视频
- 内置AI客服
- 简约扁平化风格
- 黄色为核心色，突出年轻化
- 中文呈现，无英文

**结果**：完全达到设计师水平，遵循先出设计系统再进行网页设计的逻辑。

## 五大升级

1. **全新AI原生画布**
2. **更智能的设计代理**
3. **语音交互**
4. **即时原型**
5. **完整的设计系统文档**

## 时代观点

- 学会表达要比学会动手更重要
- Vibe Design 时代已到来
- 设计门槛消失
