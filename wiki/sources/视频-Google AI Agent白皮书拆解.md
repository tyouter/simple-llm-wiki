## 概述

2025年11月Google联合Kaggle发布5篇AI Agents白皮书，系统性定义Agent概念，从系统工程与生产视角出发。

## 核心内容

### 白皮书覆盖范围

1. Agent的结构化定义
2. 工具调用与互操作协议（MCP）
3. Sessions与长期记忆
4. 质量评估与可观测性
5. 多Agent协作与生产部署

### 核心信号

- Agent != ChatBot
- Agent != Prompt工程
- Agent = 下一代软件抽象层

### 定义视角

- 把Agent当作可执行、可治理、可扩展的软件系统
- 不是Prompt技巧合集，也不是Demo教程
- 从系统工程与生产视角出发的官方定义
