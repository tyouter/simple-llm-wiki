# 开源Agent Skills社区资源整理

## 概述
开源Agent Skills社区两个月内从每周新增1100个Skills暴涨到每周产生近5万个Skills，总量突破10万个。视频介绍了如何寻找和使用好用的Skills，以及Skills MP这个独立的Skills下载站点。

## 关键内容

### 社区增长数据
- **两个月前**：每周新增开源Agent Skills约1100个
- **现在**：一周产生接近5万个Skills
- **总量**：已超过10万个开源Agent Skills
- **现象**：Skills构建者可能比用户还多

### Skills MP工具介绍
- 独立的Skills下载站点
- 整合开源Agent Skills
- 提供Skills功能描述、Stars/Fork数量、Skills包信息
- 默认按Stars排序
- 支持分类浏览

### Skills使用方法
1. 在Skills MP找到需要的Skills
2. 下载放到对应工具的Skills目录：
   - Cursor：cursor-skills目录下
   - Claude Code：claude-skills目录下
3. AI工具获得相应能力
4. 可以通过斜杠命令指定Skills

### Skills使用示例
- 使用DOCX Skills创建Word文档
- AI生成内容后调用Skills生成文档
- 年终总结自动生成Word文档
