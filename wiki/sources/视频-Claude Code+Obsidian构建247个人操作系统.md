# Claude Code+Obsidian构建24/7个人操作系统

## 概述

介绍如何用Claude Code+Obsidian构建一个24/7运行的"个人操作系统"，用于个人思考、知识积累和创业构建。

## 来源信息

- **作者**: obsidian学英语
- **发布日期**: 2026-02-24
- **时长**: 58:56
- **平台**: B站
- **观看数**: 8.2K

## 核心内容

### 用Markdown写作一切

- 日记、项目、信念、人物、会议
- 链接笔记，反映大脑真实的思考方式

### Obsidian CLI集成

- 安装Obsidian CLI让Claude Code读取整个vault
- 利用关系网络建立上下文

### 自定义斜杠命令

- `/context` - 加载完整生活和工作状态
- `/trace` - 查看想法如何在几个月中演化
- `/connect` - 桥接两个探索领域
- `/ideas` - 从vault生成创业想法
- `/graduate` - 将日常思考提升为真正资产

## 核心原则

- 人来写vault，代理读取它、建议、执行
- 从笔记内部委托任务
- 写作越多，代理拥有的上下文越多
- Markdown文件是LLM的氧气
