# GUI Agent - 嵌入网页的AI操作员

## 概述
阿里巴巴开源的GUI Agent项目，在GitHub冲上热门榜单，获得5.3K Star。这是一个能嵌入网页的AI操作员，让网站开发者可以轻松集成，让最终用户通过自然语言与网页交互。

## 关键内容

### 核心定位
- **GUI代理存在于网页中**：不同于传统浏览器自动化工具
- **面向最终用户**：不是面向开发者做爬虫或自动化任务
- **一行代码集成**：可将网站变成AI Native App

### 技术特点
- **不走Browser Extension**：不依赖浏览器扩展
- **不走Python和Headless Browser**：不需要后台自动化框架
- **基于TaxSpace DOM操作**：不靠截图OCR，使用脱水分析方式
- **高效可靠**：DOM操作方式更高效、更可靠

### 应用场景
- 老年用户轻松上手各类Web应用
- 新员工快速适应企业应用
- 任何需要简化Web交互的场景

- 冲上GitHub热门榜单
