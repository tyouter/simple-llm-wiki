# AI编程工具系统提示词开源项目

## 概述
一个开源项目揭示了主流AI编程工具背后的系统提示词，包含Cursor、Trae、Copilot、Notion AI、Complexity等30多个AI工具的提示词。项目在GitHub获得122K Star，登上GitHub上升军事榜日榜第一。

## 关键内容

### 项目特点
- 包含30多个AI编程工具的系统提示词
- 挖掘了各工具内部的提示词配置
- 提供了学习和参考案例

### 包含的工具
- Cursor（1.0到2.0版本的提示词）
- Trae（普通聊天和Builder模式提示词）
- Copilot
- Notion AI
- Complexity
- Claude Code（快速模式提示词）

### 提示词演进分析
以Cursor为例：
- **Cursor 1.0**：提示词大小不到9KB
- **Cursor 2.0**：提示词增长到37.9KB
- 随着模型能力增强，提示词内容变得更丰富

### 学习价值
- 提示词好坏直接影响AI最终效果
- 这些提示词经过无数次调试和优化
- 可作为AI应用开发的学习案例
- 解释了为什么同一模型在不同IDE中效果不同
