# Gemini CLI搭配设计Skill实战

## 概述

视频演示使用 Gemini CLI 搭配开源设计 Skill 进行 UI 设计实战。展示了如何通过 AI 编程方式快速构建专业的企业级网站界面。

## 设计案例展示

视频展示多个设计案例：

1. **企业模式网站**
   - 产品宣传页面
   - 公司机构介绍
   - 核心功能展示
   - 产品结构展示

2. **户外配件商城**
   - 户外配件售卖
   - 产品价值介绍
   - 详细结构设计

## 设计流程

- 使用设计 Skill 包快速生成页面结构
- 通过 Gemini CLI 执行设计命令
- 构建专业的 UI 设计方案
- 采用企业级设计风格

## 技术特点

- 使用 Call Code 插件
- 引用外部图片资源
- 响应式设计支持
- 前端页面设计一体化

## 设计风格

- 简洁专业的企业风格
- 商业文件级别设计
- 用户界面设备适配
- 产品展示为核心

## 工具组合

- Gemini CLI
- 设计 Skill 包
- 浏览器插件
- Call Code 工具
