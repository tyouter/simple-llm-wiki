# 把Claude Code变成12位软件专家团队

## 概述

靓仔AI进化论发布的视频，时长50秒，观看量1.1万。视频介绍YC总裁兼CEO开源的AI编程工作流——GStack，不是插件而是给Claude Code装上12个专业角色模式，让一个人就能像一个软件开发团队。

## 核心要点

### GStack角色系统

#### 规划角色

- **Plan CEO Review**: 用CEO思维讨论需求
- **Plan ENG Review**: 锁定架构、数据流和测试

#### 设计审查

- **Plan Decide Review**: 先做一轮设计审查

#### 实施角色

- **Review**: 专门抓CI能过但上线会炸的坑
- **Shift**: 负责发版推送、开PR
- **QA**: 边测边修再复测

#### 测试验证

- **Browser**: 真正进浏览器点页面看截图查报错
- **Set up Browser Cookies**: 导入真实浏览器的状态

#### 后处理

- **Retro**: 帮团队做工程复盘
- **Document Release**: 自动补齐README和发布文档

### 系统价值

- **团队级能力**: 这已经不是编程工具，是让你一个人就像一个软件开发团队
- **专业分工**: 12个角色覆盖软件开发全流程
- **开源免费**: YC总裁CEO把自己的工作流开源

### 适用场景

- **个人开发者**: 一人完成团队级工作
- **创业团队**: 用AI补充缺失的专业角色
- **Vibe Coding**: 新范式的AI编程实践
