# Oh My ClaudeCode把Claude变成Coding武器

## 概述

介绍专为Claude Code打造的"多AI编排神器"——Oh My ClaudeCode。它不再只是普通工具，而是一把真正的coding武器，包含19个专业化Agent协同作战，多种强大模式一键触发。

## 核心要点

### 多Agent架构
- Claude担任指挥官
- 平行调用Gemini（设计、视觉、文档）和Codex（架构、安全、审查）
- 19个专业化Agent：建筑师、调试器、QA测试、设计师等
- 协同作战模式

### 核心模式
- **Autopilot**：全自动从想法生成可测试代码，不需要人工干预
- **Ralph**：自循环验证架构，参照自我循环直到完成，永不停下
- **Ultrawork**：极限并行，同时处理多个任务
- **Deep Interview**：苏格拉底式深度提问理清需求，节省token
- **Team**：协调团队完成事情（开发、前端、后端、测试）
- **Planning**：基于访谈的规划，需求收集

### MCP工具集成
- LSP智能补全
- AST结构化搜索
- Python REPL数据分析
- 持久内存
- 几乎零学习成本

### 安装方法
- Claude插件市场一键安装
- 或使用npm install安装
- 两步简单配置

### 使用示例
- 完全自主运行构建待办应用
- 启动多团队执行修复任务
- 关键词触发特定模式

### 兼容性
- 支持Gemini、Claude Desktop和Claude Code
