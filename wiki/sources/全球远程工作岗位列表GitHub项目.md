# 全球远程工作岗位列表GitHub项目

## 概述

JavaPub发布的视频介绍一个GitHub开源项目（40K star），提供全球远程工作岗位列表。这不是招聘网站，而是持续更新的远程办公公司清单，帮助"逃离北上广996"的程序员找到远程工作机会，实现"数字移民"。

## 栅心要点

### 项目价值

- **数字移民签证**: 项目帮助想逃离996但不想失业的人找到远程工作机会
- **持续更新**: GitHub清单持续更新，不是静态列表
- **筛选伪远程**: 直接筛掉那些"伪远程岗位"

### 项目特点

- **公司分类**: 按国家和地区标好，方便定位
- **公司范围**: 从12rp到Automatic到GitLab、Shopify等各种远程优先科技公司
- **信息透明**: 每个公司都写清楚：是否支持远程、是否限制地区、官方招聘入口
- **一键直达**: 点链接直接到招聘页面

### 远程工作理念

- **WiFi就是工位**: 理念是"wifi就是工位，世界就是办公室"
- **全球打卡**: 在清迈写代码、在巴厘岛开会、在海边commit
- **GitHub搜索**: 搜索"remote jobs"开启全球打卡计划

### 适用人群

- **程序员**: 主要面向编程工作者
- **逃离996者**: 想离开北上广高压工作环境的人
- **数字游民**: 想实现工作地点自由的人
