# Human 3.0人生操作系统：用Gemini做个人天赋挖掘

## 概述

Human 3.0是由国外知识大神Dan Koe创建的人生规划提示词，整合了心理学、哲学、商业等多个领域的知识，帮助人们识别自己在各个领域所处的发展水平，从而在所有领域都取得进步。

## 使用方法

1. 将完整的Human 3.0提示词粘贴到Gemini
2. AI会不断向你提问
3. 按照提问回答即可
4. 大约与AI聊天20分钟后，可能会改变生活和发展的轨迹
