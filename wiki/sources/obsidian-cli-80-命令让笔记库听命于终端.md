# Obsidian CLI：80+命令让笔记库听命于终端

## 概述

## 简介


Obsidian 官方出了个 CLI，80多个命令，终于不用折腾 MCP、REST API 那堆野路子了。📌 核心卖点：
- 终端直接操作 vault：查任务、搜标签、找断链，一条命令搞定
- AI agent 的最佳搭档：Claude Code 直接调用，progressive disclosure 让 agent 自己学会怎么用
- Daily note 秒变全平台收件箱：Telegram 录条语音，agent 自动追加到今日笔记

⚡ 为什么 CLI 比 MCP 香：
更快、不用认证、支持 piping 串联操作，上下文效率拉满

🔧 上手条件：Obsidian 1。12 + Catalyst License，设置里开启即可

一句话总结：这是 Obsidian 给 AI 时代交的投名状
