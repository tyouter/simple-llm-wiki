# Stop Writing Bad CLAUDE.md Files - Claude Code配置最佳实践

## 概述
本视频由camelCase发布，详细讲解了如何编写高效的CLAUDE.md文件。大多数CLAUDE.md文件存在太长、太模糊或包含反生产力内容的问题。视频基于真实LLM研究，提供了8个关键技巧来优化Claude Code的性能。

## 关键内容

### CLAUDE.md常见问题
- 文件太长
- 内容太模糊
- 包含反生产力内容

### 8个关键技巧

#### Tip 1: Size Matters（文件大小很重要）
- 控制CLAUDE.md文件大小，避免过长

#### Tip 2: Don't use /init（不要使用/init）
- 避免使用/init命令生成配置

#### Tip 3: No codestyle（不要包含代码风格）
- 代码风格相关内容可能干扰AI理解

#### Tip 4: Start with these 3 things（从这三个内容开始）
- 确定核心配置起点

#### Tip 5: Progressive disclosure（渐进式披露）
- 分层次提供信息，而非一次性全部展示

#### Tip 6: Why Claude ignores you（为什么Claude忽略你的指令）
- 理解AI忽略指令的原因和解决方案

#### Tip 7: Keep it updated（保持更新）
- 定期更新CLAUDE.md文件

#### Tip 8: Different location options（不同位置选项）
- 了解CLAUDE.md文件的不同放置位置选项

### 相关配置文件
- CLAUDE.md适用于Claude Code
- AGENTS.md适用于其他AI工具
- 也适用于Cursor、Codex等AI编程工具
