# CanIRun本地大模型评估工具

## 概述

鲲鹏Talk介绍的CanIRun.ai网站，帮助用户快速评估本地硬件能否运行各类大模型，30秒出结果，提供S/A/B/C/D/F评分体系。

## 关键内容

### 核心功能
- 浏览器直接测GPU、显存、内存
- 30秒出结果
- 每个热门模型能否跑、跑得顺的评估
- 支持Q4_K_M量化评估
- S/A/B/C/D/F评分体系

### 安全特点
- 零上传、纯本地计算
- 隐私保护

### 网站地址
https://www.canirun.ai/

### 评估示例
- Mac M3 Max 32G评估
- RTX 5090、5070、4090评估
- H100、A100企业级显卡评估

### 推荐模型
- Qwen 3.5 27B和35B
- 参数小但能达到较好推理速度

### 硬件与速度示例
- M3 Max 32G: Qwen 3 32B约16 token/秒
- RTX 5090: Qwen 3.5 9B约88 token/秒
- RTX 4090: Qwen 3 35B约39 token/秒
