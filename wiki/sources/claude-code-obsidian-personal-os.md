# Claude Code + Obsidian 构建24/7个人操作系统

## 概述

本视频介绍如何将 Claude Code 与 Obsidian 结合，构建一个"24/7个人操作系统"，用于个人思考、知识积累和创业构建。

## 核心方法论

### 用 Markdown 写作一切

- 日记、项目、信念、人物、会议均用 Markdown 记录
- 链接笔记，让它们反映大脑真实的思考方式

### Obsidian CLI 集成

安装 Obsidian CLI，让 Claude Code 能读取整个 vault 及其关系网络，避免每次会话重新解释项目。

### 自定义斜杠命令

构建以下自定义命令：

| 命令 | 功能 |
|------|------|
| `/context` | 加载完整生活和工作状态 |
| `/trace` | 查看想法如何在几个月中演化 |
| `/connect` | 桥接两个一直在探索的领域 |
| `/ideas` | 从 vault 中生成创业想法 |
| `/graduate` | 将日常思考提升为真正的资产 |

### 核心规则

**人来写 vault，代理读取它、建议、执行**

- Claude (Clode) 揭示多年来无意识循环的模式
- 从笔记内部委托任务：在 Obsidian 里写一句话，代理处理剩下的
- 写作是杠杆：写得越多，代理拥有的上下文就越多

## 关键洞察

**Markdown 文件是 LLM 的氧气**

通过持续的 Markdown 写作，为 AI 提供丰富的上下文，使其能够理解个人思考模式并提供精准建议。
