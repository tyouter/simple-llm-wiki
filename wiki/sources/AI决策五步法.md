# AI决策五步法

## 概述

作者将常用的一套AI决策流程封装成 Skill，开源到 GitHub。核心观点：AI真正改变的不是效率，而是决策质量。

## 核心观点

- AI更大的价值是帮助提升判断和决策能力
- 不是替你做决定，而是帮你把复杂问题拆开、看清、推演
- AI时代，执行会被替代，但判断和决策越来越值钱

## 五层决策结构

### 第一层：信息层

**目的**：收集行业信息
- 了解外部市场
- 了解行业情况
- 使用 Perplexity 进行搜索性调研
- 快速拉取行业数据、趋势、案例
- 避免建立在错误信息基础上

**要点**：决策第一步最怕的不是复杂，而是信息错

### 第二层：结构层

**目的**：行业信息与自身信息对比分类
- 使用 Claude 进行逻辑推演
- 发现你没问出来的问题
- 帮你去发现你没有想到的问题

**要点**：人类做决策最容易犯的是主观认识偏差

### 第三层：推理层

**目的**：多路径推演和决策模拟
- 使用 ChatGPT 展开多种不同路径
- 未来可能性推演
- 决策的关键是多种可能性及判代价

**要点**：GPT擅长判断和模拟型推演

### 第四层：评估层

**目的**：决策评分量化
- 使用 Notion 做评分系统
- 感性判断结构化
- 可对比评分系统
- 持续迭代的决策系统

**要点**：让答案变成可量化的分析和对比

### 第五层：呈现层

**目的**：结果呈现
- 表达与决策推演不同结构
- 用专业AI系统做表达
- 得到完整的决策风险地图

**要点**：决策是决策，推演是推演，表达是表达

## 工具选择逻辑

- **Perplexity**：搜索性调研，获取真实数据
- **Claude**：逻辑推演，发现问题
- **ChatGPT**：多路径推演和模拟
- **Notion**：评分量化，持续迭代
- **Gamma**：最终表达呈现

## 应用场景

- 是否全身投入某项目
- 创业赛道选择
- 是否读博或换城市
- 企业和个人重大决策
