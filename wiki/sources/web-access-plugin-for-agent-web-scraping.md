# web-access插件解决Agent网页抓取痛点

## 核心痛点

本地跑Agent抓网页常见问题：
- 动不动弹个浏览器抢你鼠标
- 遇到登录框直接卡死
- 验证码处理困难
- 反爬虫检测导致无法访问

## web-access插件解决方案

### 核心功能
- 直接使用本地浏览器的登录状态访问网站
- 继承已有cookies和session
- 验证码直接过
- 上百个网页全在后台静默跑
- 绝不打断敲代码工作流

### 反爬绕过能力
- 踩过反爬的坑会记下来绕道走
- 自动学习反爬策略
- 减少手动处理反爬的麻烦

## 使用场景

适合需要大量网页抓取的Agent任务：
- 数据采集
- 网站内容分析
- 自动化测试
- 需登录态的网页操作
