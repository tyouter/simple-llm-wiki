# Product Manager Skills：让Claude秒变产品经理的Skill

## 项目概述

Product Manager Skills是一个Claude Code Skill，由资深产品经理开源在GitHub上，包含46个产品管理框架，让Claude像读过这些方法论的专业PM一样工作。

## 核心内容

- **46个产品管理框架**：完整的产品方法论体系
- **方法论拆解**：将专业框架拆解为Claude能理解的模块
- **主动引导工作流**：Claude会按框架主动引导用户工作

## 框架示例

- 优先级排序（RICE模型）
- 用户洞察分析
- 用户故事撰写
- 需求池管理
- 产品定位梳理

## 使用演示

输入5个产品需求，让Claude做RICE优先级排序：
- 按Reach、Impact、Confidence、Effort四个维度逐个打分
- 给出综合排序结果
- 告知哪些是Quick Wins、哪些是Big Bets

## 多角色适用

| 角色 | 适用场景 |
|------|---------|
| 产品经理 | 专业工作流执行 |
| 运营 | 用户洞察分析 |
| 创业者 | 产品定位梳理 |
| 项目经理 | 需求池管理 |

## 本质定位

本质上是一套决策框架，适用于任何需要做需求分析的场景。
