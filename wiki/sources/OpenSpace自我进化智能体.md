# OpenSpace自我进化智能体

## 概述

香港大学实验室开源的项目 OpenSpace，能让本地安装的 AI Agent（如 Claude Code、Codex、Open Code 等）实现自我进化。

## 核心功能

### 自我进化机制

在AI开发过程中不断总结经验，自动提取多种 Skills 技能：
- 先生成初始技能
- 累计生成70+新技能
- 后续任务消耗的 Token 数量降低

### 三种进化模式

1. **修复模式**：修复损坏或过时的指令
2. **增强模式**：基于负技能创建增强版或专业版技能
3. **提取模式**：从成功案例中提取可复用技能

### 持续监测

- 监测技能表现
- 发现落后或表现不佳的技能
- 强化或替换

## 工作流程示例

以数据看板开发任务为例：
1. 分析开源项目
2. 生成6种初始技能
3. 项目开发过程累计70+新技能
4. 后续类似任务执行效率提升

## 使用方式

### 方式一：集成到本地Agent

- 部署到本地
- 集成到 Claude Code 等 Agent
- Agent 自动进化

### 方式二：命令行直接使用

- 部署到本地
- 通过命令行使用 OpenSpace
- 不依赖其他代理应用

## 社区功能

- 技能自己进化
- 将进化中的技能分享给其他人
- 使用其他人进化出来的智能体技能

## 测试数据

港大用50个任务作为测试数据，后续任务各方面都取得进步。
