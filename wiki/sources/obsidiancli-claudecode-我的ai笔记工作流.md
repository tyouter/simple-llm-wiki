# ObsidianCLI+ClaudeCode:我的AI笔记工作流

## 概述

## 简介


Obsidian 官方推出了 CLI 命令行工具，面向所有用户开放。这意味着 AI 可以直接调用你的笔记、日志、标签、反向链接——不再需要逐个文件扫描。这期视频分享我用 Obsidian CLI + Claude Code 搭建的真实工作流：
1、每日开场：一条指令，AI 自动加载记忆、读取日志、查看任务，帮我规划一天重点
2、Skill 系统：把常用操作封装成可复用的工作流，随时调用
3、演示：如何让 AI 帮你从零创建一个 Skill，全程不用写代码
CLI 带来两个核心变化：数据读取更快更准，能力边界被彻底打开
