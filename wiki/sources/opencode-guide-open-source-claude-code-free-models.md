# OpenCode详细攻略：开源版Claude Code免费模型与神级插件

## 概述

OpenCode 是近期热度最高的 AI 编程工具，最大优势是开箱即用的免费模型。通过简单配置，还能免费接入 Gemini Pro、Claude 4.5 Opus 等顶级编程模型。OpenCode 可看作开源版的 Claude Code，几乎具备 Claude Code 所有功能。

## 核心要点

### OpenCode四种形态

- 开箱即用的免费模型版本
- 可配置接入 Gemini Pro
- 可配置接入 Claude 4.5 Opus
- 支持多种模型切换

### 核心功能

- 几乎具备 Claude Code 所有功能
- 支持 Agent Skills
- 支持 MCP
- 支持 SubAgent 等高级特性

### Oh My Open Code插件

- 超强插件扩展
- 提供更多高级用法
- 增强工具能力

### 适合人群

- 小白上手 AI 编程
- 可以随便造、随便玩
- 练习 Agent Skills、MCP、SubAgent
- 无成本学习高级特性

### 配置亮点

- 免费接入顶级编程模型
- 无需付费订阅
- 本地运行，数据安全
