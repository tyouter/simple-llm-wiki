# Claude Code职场替身应用

## 概述

珍妮丁丁AI说发布的视频，分享非技术人员如何将Claude Code作为职场替身使用，展示了Claude Code在办公场景中的两大核心能力：职场狗腿子机制和自我迭代合作机制。

## 关键内容

### 第一绝：职场狗腿子

Claude Code展现的职场分析能力：
- 自动分析每个领导的性格、立场和期待
- 写汇报时告诉不同领导想看什么内容
- 建立领导的画像和背后的沟通记录
- 根据领导画像调整汇报内容和策略

### 第二绝：自我迭代合作机制

Claude Code建立的动态更新系统：
- 对话中发现产品方向和业务变化，实时记录
- 更新进基础知识库
- 写周报思考本周获得的经验教训
- 思考以后如何更好地服务

### 基础知识库结构

Claude Code维护的工作文档包含：
1. 工作记录
2. 工作模式与偏好
3. 实时更新的业务信息

### 适用人群

- 需要写汇报/规划的职场人
- 想让AI真正了解工作全貌的人
- 受够了每次和AI重新解释背景的人

### 方法论核心

这是底层方法论，任何编程工具都可以这样玩，关键在于CLAUDE.md文档的编写。
