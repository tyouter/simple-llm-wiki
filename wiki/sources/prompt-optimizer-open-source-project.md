# 提示词优化器开源项目介绍

## 项目概况

- 登上GitHub Trending榜单
- 已获得2万+点赞
- 傻瓜式操作生成专业提示词

## 核心功能

### 使用场景
当需要一段提示词但只有零星模糊想法时：
- 用户描述大致需求（如"去除AI味"）
- 优化器自动生成专业提示词
- 不到一分钟完成

### 支持模式
- 文生图
- 图生图
- 其他多种模式

### 跨平台支持
- Windows
- Linux
- macOS

## 核心价值

解决提示词写作痛点：
- 用户想法模糊但知道大致方向
- 自动将模糊想法转化为专业表述
- 降低提示词写作门槛
