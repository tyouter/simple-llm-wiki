# Vibe Coding AI资讯站点

## 概述

李自然春节期间通过 Vibe Coding 方式制作的两个AI资讯网站，本来自用，现已公开。主打"不加味精"的信息筛选理念。

## 问题背景

中文AI资讯存在的问题：
- 对英文信息的二手加工
- 标题党流行：炸裂、杀疯了、赢麻了
- 夸张串联弱证据讲大故事
- 相当多内容是自媒体写的软文推广

## 站点一：AI资讯速览

**地址**：https://ai-digest.liziran.com

**特点**：
- 每天从20+一手英文信息源筛选
- 行业新闻不夸张不接软文
- 保证接近真实的信息材料
- 每天约5篇主要文章+几条快讯

**定位**：比别人更浅，但更接近真实

## 站点二：AI论文简报

**地址**：https://ai-brief.liziran.com

**特点**：
- 每天从arXiv近300篇论文筛选3-5篇
- 不需要机器学习背景也能看懂
- 非技术人员的简本解读
- 了解AI前沿技术动态

**定位**：从论文源头把握AI前沿

## 目标受众

- 产品经理
- 创业者
- 独立开发者
- AI行业从业人员

## 获取方式

1. **网站直接看**：适合每天看网站习惯
2. **Newsletter订阅**：网站本身提供订阅
3. **RSS订阅**：RSS也可用
4. **微信公众号**：同步更新到「李自然说」公众号

## 技术实现

- 春节期间 Vibe Coding 方式制作
- 使用某种自动化流程
- 需要作者扫码发布公众号内容
- 网站北京时间早上8点多更新
