# DeerFlow 2.0：字节开源深度研究多智能体框架

## 概述

字节跳动开源的 DeerFlow 2.0 已获得超过 2 万 GitHub 星标。这是一个深度研究型多智能体框架，不是 1.0 的升级版，而是从零开始的全新版本，能帮助用户完成深度调研、网页搜索、内容分析、代码执行、内容生成等几乎任何任务。

## 核心要点

### 核心能力

- **深度调研**：自动搜索网页、分析内容
- **多智能体协作**：内置多智能体系统，分工合作、并行执行
- **记忆偏好**：能记忆用户偏好，让每次请求更聪明高效
- **代码执行**：支持代码生成与执行
- **内容生成**：自动生成各类内容

### 技术特性

- 支持各种主流搜索引擎集成
- 支持自定义工具扩展
- 支持私有知识库接入
- 全流程智能解决方案

### 应用场景

- 学习科研自动化
- 办公自动化
- 深度调研分析
- 复杂任务编排
