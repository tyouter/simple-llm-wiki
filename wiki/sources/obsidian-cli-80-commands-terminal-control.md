# Obsidian CLI：80+命令让笔记库听命于终端

## 概述

Obsidian 官方发布了 CLI（命令行界面）工具，提供 80 多个命令，让用户可以在终端直接操作笔记库。这是 Obsidian 给 AI 时代交的"投名状"，让笔记库能被程序优雅地操控，成为 AI Agent 的最佳搭档。

## 核心要点

### 核心卖点

- **终端直接操作 vault**：查任务、搜标签、找断链，一条命令搞定
- **AI agent 最佳搭档**：Claude Code 直接调用，progressive disclosure 让 agent 自己学会怎么用
- **Daily note 秒变全平台收件箱**：Telegram 录条语音，agent 自动追加到今日笔记

### CLI 相比 MCP 的优势

- 更快、不用认证
- 支持 piping 串联操作
- 上下文效率拉满
- 无需折腾 REST API 等野路子

### 上手条件

- Obsidian 1.12 版本以上
- Catalyst License
- 设置里开启 CLI 功能即可

### 实际应用场景

- Claude Code 通过 CLI 操作 Obsidian vault
- 自动创建、修改、搜索笔记
- 实现笔记库的自动化管理
- 与 AI Agent 深度集成，让 AI 能"理解"笔记结构
