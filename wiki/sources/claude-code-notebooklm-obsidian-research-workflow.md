# Claude Code + NotebookLM + Obsidian 研究工作流

## 概述

视频介绍了一个实用、可重复的工作流程，将 Claude Code 转变为一个完整的研究代理——搜索 YouTube，通过 NotebookLM 进行深度分析，生成交付成果，并将所有内容组织在 Obsidian 知识库中。全部设置过程不超过30分钟。

## 核心要点

### 三者结合的力量

- **Claude Code**：免费的价值，AI 编程与任务执行
- **NotebookLM**：令人惊叹的深度分析能力
- **Obsidian**：知识库管理与组织

### 工作流程

1. Claude Code 搜索并获取 YouTube 视频内容
2. NotebookLM 对内容进行深度分析和理解
3. 生成交付成果（总结、报告等）
4. Obsidian 组织和存储所有知识

### 实际应用

- 研究任何需要分析的内容
- 创建任何需要的交付成果
- 自动化知识管理流程
- 30分钟内完成完整设置

### 核心价值

这不是理论，而是作者实际使用的系统，可以直接采用并应用于任何研究需求。
