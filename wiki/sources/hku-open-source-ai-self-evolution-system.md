# 港大开源AI自我进化系统

## 核心特点

让AI越用越省钱的同时还越来越聪明：
- 自我监控系统盯着AI每一步动作
- 报错立马捕获
- 自动复盘失败原因

## 工作机制

### 自动学习循环
1. 监控AI每一步动作
2. 报错时捕获失败信息
3. 将报错代码丢给大模型重新思考
4. 修改好的代码封装成标准技能
5. 存进技能库

### "吃一堑长一智"
真正让AI学会积累经验：
- 每次失败都转化为学习机会
- 失败经验标准化存储
- 后续执行避免同类错误

## 效果验证

经测试发现：
- 循环几轮后token消耗降低46%
- AI越来越懂用户
- 形成个性化的AI系统

## 核心洞见

- 让AI越用越聪明的同时还能省钱
- 可以"养一个属于自己的AI系统"
- 经验积累带来效率提升和成本下降
