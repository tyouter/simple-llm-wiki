# 个人操作系统

## 定义

个人操作系统是一种将个人知识、思考、工作和生活整合为一个系统化框架的概念，通过 AI 和笔记工具实现24/7的智能辅助。

## 核心理念

### Markdown 作为核心

**Markdown 文件是 LLM 的氧气**

通过持续的 Markdown 写作，为 AI 提供丰富上下文，使其能够理解个人思考模式并提供精准建议。

### 人机协作分工

- **人**: 写 vault，进行创意和决策
- **AI**: 读取 vault，提供建议和执行

### 链接网络

笔记之间的链接反映大脑真实的思考方式，构建知识网络。

## 实现方法

### Obsidian CLI 集成

让 Claude Code 能读取 vault 及其关系网络，避免每次会话重新解释项目。

### 自定义斜杠命令

| 命令 | 功能 |
|------|------|
| `/context` | 加载完整生活和工作状态 |
| `/trace` | 查看想法如何在几个月中演化 |
| `/connect` | 桥接两个正在探索的领域 |
| `/ideas` | 从 vault 中生成创业想法 |
| `/graduate` | 将日常思考提升为真正的资产 |

### 从笔记内委托任务

在 Obsidian 里写一句话 → Agent 处理剩下的。

## 应用价值

- 个人思考的可追溯性
- 知识积累的系统化
- 创业构建的辅助决策
- 模式识别与自我洞察
