**Beta Distribution via Plugin Manager (BRAT)** refers to the specialized distribution channel for pre-release Obsidian plugins using the "Beta Reviewer's Auto-update Tool." This system allows developers to share in-development plugins directly from GitHub repositories, bypassing the official community plugin marketplace's requirement for stable, production-ready releases.

BRAT serves as critical infrastructure within the [[Obsidian Plugin Ecosystem]], enabling rapid iteration and early user feedback. For plugins like [[Obsidian Dragger]], which are in active development, BRAT provides a safe installation method that maintains auto-update functionality. Users can test cutting-edge features while developers gather real-world usage data before committing to a stable release.

This distribution model lowers barriers for both developers (who can share work-in-progress) and users (who can access innovations earlier). It represents a pragmatic solution to the tension between quality control and innovation velocity in community ecosystems. By facilitating beta testing, BRAT helps ensure that plugins entering the official marketplace have been vetted by engaged community members.
