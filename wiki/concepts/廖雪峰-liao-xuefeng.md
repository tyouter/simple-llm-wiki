# 廖雪峰 (Liao Xuefeng)

国内知名技术教育博主，以 Python 教程闻名，其教程风格通俗易懂、实践性强。

## 核心要点

- **Python 教程**: 撰写国内最受欢迎的 Python 入门教程，以在线形式持续更新
- **通俗教学**: 教程风格简单易懂，配合实例代码，降低编程入门门槛
- **在线教程模式**: 采用网站而非书籍的形式发布教程，保持内容的实时更新
- **广泛影响**: 教程成为大量国内程序员 Python 学习的第一站
- **技术博客**: 在个人网站持续分享技术见解和编程心得
