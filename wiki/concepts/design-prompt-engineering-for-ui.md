# Design Prompt Engineering for UI

## 概述

设计提示工程是专门针对UI/界面生成的Prompt工程方法，通过结构化的设计规范(XML格式)来指导AI生成高质量、无AI感的界面代码。

## 核心要点

- **XML设计规范**：使用结构化XML格式定义设计系统、配色、排版等规范
- **风格多样性**：涵盖Brutalist、Retro、Minimalist等多种设计风格
- **AI感规避**：通过专业设计提示避免Tailwind CSS等典型AI生成特征
- **预构建库利用**：依赖designprompts.dev等预构建设计提示库而非从头编写
- **合成与适配**：从多个设计提示中选择、组合、适配为项目定制规范
