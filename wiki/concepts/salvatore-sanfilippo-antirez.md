# Salvatore Sanfilippo (antirez)

Redis 数据库的创造者，以追求极致性能和简洁代码的工程哲学闻名于开源社区。

## 核心要点

- **Redis 创造者**: 2009年创造了 Redis，这个内存键值存储系统成为现代 Web 应用的基础设施组件
- **性能优先**: 坚持 C 语言实现，追求极致性能和资源效率，Redis 成为缓存系统的行业标准
- **简洁代码哲学**: 以"代码应该像散文一样可读"为理念，Redis 源码被誉为教科书级别的清晰代码
- **个人博客**: 在 antirez.com 分享技术见解和创业思考，文章风格深入、坦诚
- **开源精神**: 期独立维护 Redis，体现了个人开发者对开源项目的深度投入
