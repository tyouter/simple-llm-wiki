# Systematic Learning Framework

Sunny 提出的战略管理学习框架，采用三层递进结构（基础→扩展→应用）系统化地构建跨学科专业知识。

## 核心要点

- **三层递进结构**: 基础层（核心理论）→ 扩展层（职能知识）→ 应用层（实践框架）
- **核心理论层**: 包括 Porter Trilogy 等战略管理经典著作，奠定理论基础
- **职能扩展层**: 涵盖研发、营销、财务、运营等功能领域的知识整合
- **实践应用层**: Balanced Scorecard、Strategy Map 等落地框架的学习
- **跨学科整合**: 强调经济学、政治学、社会学、心理学等多学科知识融合
