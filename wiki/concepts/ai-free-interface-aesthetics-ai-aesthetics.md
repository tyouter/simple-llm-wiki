# AI-Free Interface Aesthetics / AI aesthetics

## 概述
无AI感界面美学是指通过特定技术和工作流程，使AI生成的界面避免可识别的AI视觉特征，呈现独特、个性化的设计风格。

## 核心要点
- **AI刻板特征**：可识别的AI视觉特征包括通用渐变、过度使用的Tailwind CSS模式、典型的"Claude味"
- **设计Prompt库**：使用策展的提示词库（designprompts.dev）提供超过20种预览设计风格，如Brutalist、Retro、Minimalist
- **XML格式Prompt**：使用XML格式的设计规范文件定义视觉风格和组件规则
- **合成优于复制**：通过组合和适配多个提示词创建定制化设计方向，而非直接复制
- **工具链协作**：结合多个AI工具（Claude Code + Gemini/Antigravity）优化前端设计输出
