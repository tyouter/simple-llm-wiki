## 简介


AI Agent 虽然聪明，但联网获取信息一直是痛点——平台反爬、API收费、跨域封禁，每上一个平台就要踩一遍坑。Agent-Reach 正是为解决这一痛点而来：它把 X、Reddit、YouTube、B站、小红书、抖音、微信公众号等 17 个主流平台的搜索、读取能力，打包成开箱即用的 CLI 工具，已在 GitHub 收获 10.4k Stars。
安装只需一句话，Cookie 仅存本地保障隐私，支持 Claude Code、OpenClaw、Cursor 等主流 AI Agent。无论搜推特、看 B 站、读公众号，Agent-Reach 都能让你的 AI 真正&quot;看见&quot;互联网。
